Quicksearch the Admin Menu and Plugin Tables.

== Description ==

Quicksearch the Admin Menu and Plugin Tables. Zero-Configuration-Plugin, just activate and search boxes are available. Javascript Required.

Development of this Plugin is sponsored in part by [Semiologic Pro](http://www.semiologic.com/).

= Features =

*   Searches instantly while you type.
*   Automatically opens matching main menus.
*   Search Term Highlighting
*   Searches [Plugin-Tables](http://codex.wordpress.org/Plugins_Installed_SubPanel).
*   Searches [Semiologic Plugin Table in Version Checker](http://www.semiologic.com/software/version-checker/).
*   Free

== Screenshots ==

== Installation ==

This Plugin can be used as a standard Plugin. Please refer to your products documentation how to install plugins or contact your site administrator. Javascript is required in your Browser.
